### Purpose:

This tool helps the Ad Ops team stay up-to-date with the various VAST and 1x1 macros that vendors 
provide.


### To use:

Download the directory.

```bash
npm install

# In one terminal tab:
npm run dev

# In another terminal tab:
npm run server
```

Then, open a browser and go to the localhost url shown in your "npm run dev" tab.



### To develop:

After cloning this directory, feel free to continue development.

Be sure to ask Claude to update the "dev_docs.md" file after a coding session.
